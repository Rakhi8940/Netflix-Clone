# Netflix Clone

A web-based Netflix clone created using **HTML**, **CSS**, and **JavaScript**. This project replicates the basic structure and functionality of Netflix, allowing users to browse through movie and TV show categories, view details, and play trailers.

## Features

- Responsive layout with a Netflix-style interface
- Browse through movie and TV show categories like "Trending Now", "Top Rated", and "Upcoming"
- Display movie/poster thumbnails and titles in a grid view
- View movie details including a trailer (using embedded YouTube videos or similar)
- Basic navigation between different sections (Home, Movies, TV Shows)
- Responsive design that works on both desktop and mobile devices

## Technologies Used

- **HTML**: Used for structuring the page, creating the layout for movies, TV shows, and other elements.
- **CSS**: Used for styling the user interface, including the grid of movies, navigation bar, and overall layout.
- **JavaScript**: Used for handling dynamic content like displaying movies and TV shows from a mock database (or real API) and handling UI interactions.
